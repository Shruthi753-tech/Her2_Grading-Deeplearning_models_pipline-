# Model Weights Directory

## ConvNeXt-Tiny Weights

Place your ConvNeXt-tiny checkpoint file here:
- **File**: `convnext_tiny.ckpt`
- **Purpose**: Tumour region classification in preprocessing
- **Download**: [ConvNeXt Model Zoo](https://github.com/facebookresearch/ConvNeXt)

## Class Weights

Auto-generated class weights will be saved here:
- **File**: `class_weights.csv`
- **Generated by**: `dataset_stats.py`
- **Format**: CSV with Class,Weight columns

## Training Checkpoints

Trained model checkpoints will be saved in:
- `../models/unet_dca_lka_fold_*.ckpt` - Segmentation models
- `../models/mil_classifier_fold_*.ckpt` - MIL models

## Usage

The pipeline will automatically:
1. Check for existing weights
2. Download if needed (future feature)
3. Generate class weights from dataset
4. Save trained models after each fold
